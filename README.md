## Sobre mim

Developer autodidata, focado em desenvolvimento **full-stack** com TypeScript e Python. Construo aplicações web, APIs e automações — do frontend em React/Next.js ao backend em Bun/Node.js/FastAPI, com deploy em Docker e VPS Linux.

Trabalho com integrações de APIs externas (Twilio, Gmail OAuth, Notion, Google Calendar) e sistemas de IA (Claude, OpenAI) como parte do stack, mas o foco é **engenharia de produto end-to-end**: código limpo, infraestrutura sólida, soluções que resolvem problemas reais.
